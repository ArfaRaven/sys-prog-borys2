Коротка інформація

GNU/Linux — це потужна операційна система з відкритим вихідним кодом, яка популярна завдяки своїй надійності та свободі. Вона використовується як на серверах, так і на робочих станціях і дозволяє користувачам мати повний контроль над своїм середовищем.

Найпопулярніші дистрибутиви

Ubuntu

Зручний для новачків і використовується в бізнесі.

Fedora

Для тих, хто прагне нових технологій і стабільності.

Arch Linux

Максимальна гнучкість для досвідчених користувачів.

Debian

Стабільний та надійний вибір для серверів.

Основні команди Linux CLI

Команда Опис
pwd Виводить поточну директорію.
ls Перегляд файлів у директорії.
cd Переміщення між директоріями.
touch Створення нового файлу.
mkdir Створення нової директорії.
cp Копіювання файлів і папок.
mv Переміщення або перейменування файлів.
rm Видалення файлів.
history Перегляд історії команд.

Інтерактивний BASH-скрипт


#!/bin/bash

# Авторизація
correct_user="admin"
correct_password="password123"

echo "Введіть ім'я користувача:"
read user

echo "Введіть пароль:"
read -s password

if [[ $user == $correct_user && $password == $correct_password ]]; then
    echo "Вітаємо, $user!"
else
    echo "Невірні дані!"
fi
